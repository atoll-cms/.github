# atoll-cms

Modern flat-file CMS for PHP with island architecture, admin UI, and updater-ready core.

## Repositories

- [atoll-core](https://github.com/atoll-cms/atoll-core) - core runtime, admin, migrations, default theme
- [atoll-starter](https://github.com/atoll-cms/atoll-starter) - template repository for new projects
- [atoll-docs](https://github.com/atoll-cms/atoll-docs) - concept and operational documentation
- [atoll-website](https://github.com/atoll-cms/atoll-website) - public landing page
- [atoll-updates](https://github.com/atoll-cms/atoll-updates) - update manifests for remote core updates

## Quick start

Use the **atoll-starter** template and run:

```bash
composer install
php bin/atoll serve 8080
```
